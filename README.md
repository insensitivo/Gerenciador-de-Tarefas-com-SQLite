# Gerenciador de Tarefas com SQLite

Projeto em Python utilizando SQLite para persistência de dados

## Tecnologias

 Python
 SQLite
 Estrutura modular

## Funcionalidades

 Criar tarefas com título e descrição
 Listar tarefas
 Marcar como concluída
 Remover tarefas
 Persistência em banco de dados

## Execute

python main.py
